rit.ai
======

### Requirements
[https://bower.io/] (Bower)
[https://www.npmjs.com/] (npm || yarn)
[https://gulpjs.com/] (gulp)
[https://github.com/purescript-contrib/pulp](Pulp)

### Installation

bower install && (npm || yarn) install 

### Running the server

pulp build && pulp run -m Server


Should display:

    * Build successful.
    Listening on 8080
